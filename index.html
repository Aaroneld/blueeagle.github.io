Moving forward!

January 16, 2020 by Collin Ballou

It has been a while since I've even looked over this site. To be honest, I was sort of cringing at the lack of dynamics offered by the static site hosting it is platformed on. See, I hosted this on Github Pages the day I discovered Pages exist. If I had known from the beginning that it only allowed me to use one page for the whole site and it lacks a number of basic features I would have decided to go with a more established host. I am currently working on building a replacement for this site that I will have hosted elsewhere and I'm thinking to use Flask to build templates for it. This site was built almost from scratch excepting some imports from bootstrap for style. I'm not stylish, at least Bootstrap is!

Fully Engaged In a Good Cause!

Ferbruary 8, 2020 by Collin Ballou

Engaging myself has been something I've learned over time. As someone who has a natural desire to learn in an ever more crowded world, I have had to learn to find sanctuary. I've noticed certain things that help me pull away from distractions in the world and focus intently on finishing each task I have in front of me. First and foremost is the things that run through my head constantly When I set out to accomplish something I find myself writing a list of all things that consume my brain power and prevent me from giving my all. Just as in the scripture D&C 88:67-68, we can chose to allow ourselves to be consumed in ignorance, or devote our whole selves to aquiring and utilizing knowledge for betterment.
