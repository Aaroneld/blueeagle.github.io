Moving forward!

January 16, 2020 by Collin Ballou

It has been a while since I've even looked over this site. To be honest, I was sort of cringing at the lack of dynamics offered by the static site hosting it is platformed on. See, I hosted this on Github Pages the day I discovered Pages exist. If I had known from the beginning that it only allowed me to use one page for the whole site and it lacks a number of basic features I would have decided to go with a more established host. I am currently working on building a replacement for this site that I will have hosted elsewhere and I'm thinking to use Flask to build templates for it. This site was build almost from scratch excepting some imports from bootstrap for style. I'm not stylish, at least Bootstrap is!
